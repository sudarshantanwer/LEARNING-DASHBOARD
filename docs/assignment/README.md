# Junior developer assignment (AI coding lab)

This folder contains the **guided rebuild lab** for learning AI-assisted development.

| File | Purpose |
| --- | --- |
| [`PROJECT_UI_WALKTHROUGH.md`](./PROJECT_UI_WALKTHROUGH.md) | **Single-file project tour:** all screen mockups embedded + every UI building block and its role |
| [`PROJECT_UI_WALKTHROUGH.pdf`](./PROJECT_UI_WALKTHROUGH.pdf) | **Printable PDF** of the same walkthrough (images + text); regenerate with [`../../scripts/generate-walkthrough-pdf.py`](../../scripts/generate-walkthrough-pdf.py) |
| [`AI_CODING_LAB.md`](./AI_CODING_LAB.md) | Full step-by-step instructions, prompts, checkpoints, and rubric |
| [`UI_MOCKUPS.md`](./UI_MOCKUPS.md) | Compact gallery of the same images + routes (optional if you use the walkthrough) |
| [`ui-mockups/`](./ui-mockups/) | PNG mockups per screen |
| [`expected-dashboard-ui.png`](./expected-dashboard-ui.png) | Dashboard-only image (same art as dashboard in `ui-mockups`) |

Start with **AI_CODING_LAB.md** to build the app; use **PROJECT_UI_WALKTHROUGH.md** to understand how the UI fits together.
